## Contributing

Contributions are welcome and appreciated.

### What you can contribute

- Bug reports and fixes
- Performance improvements
- New metrics or exporters
- Documentation improvements
- Tests and stability improvements
- Integration with other tools (Grafana dashboards, alerts, etc.)

---

### How to contribute

1. Fork the repository
2. Create a new branch:
   ```bash
   git checkout -b feature/your-feature-name
   ```
3. Make your changes
4. Ensure the project builds and works correctly
5. Commit your changes with a clear message:
   ```bash
   git commit -m "feat: add new traffic metric"
   ```
6. Push to your fork and open a Pull Request

---

### Guidelines

- Keep code clean and readable
- Follow existing project structure and conventions
- Add comments where necessary
- Update documentation if you change behavior
- Prefer small, focused PRs instead of large ones

---

### Notes

- For major changes, please open an issue first to discuss what you would like to change
- If you are unsure about anything — feel free to ask

---

### Support

If you find this project useful, consider giving it a star — it helps a lot!
